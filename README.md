 ## Description
 
 Search for the cheapest flight in a 6-month range departing & arriving to the user chosen destination. Alerts can be received via text or email.
 
 ### Features
 
 - Alert for pricing sent directly to your phone or email, no need to search endlessly through travel websites
 
 ### Built with
 
 - Python
 - Flask
 - Jinja
 - Bootstrap
